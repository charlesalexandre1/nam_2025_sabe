📊 Como interpretar os gráficos de pizza

Os gráficos abaixo representam a distribuição percentual média dos alunos por nível de proficiência em cada disciplina, considerando todas as séries da escola no ano de {{ ultimo_ano }}. Cada fatia corresponde a um nível:

Abaixo do Básico – alunos que não atingiram o mínimo esperado. Adequado – alunos que demonstram o aprendizado esperado.
Básico – alunos com domínio parcial, mas ainda insuficiente. Avançado – alunos que superam as expectativas.

Quanto maior a fatia verde/azul, melhor o desempenho na disciplina. {% if dados_grafico|length >= 2 %} Por exemplo, em {{ dados_grafico.0.disciplina }} a maior concentração está no nível {% if dados_grafico.0.adequado > dados_grafico.0.avancado %} Adequado ({{ dados_grafico.0.adequado }}%) {% else %} Avançado ({{ dados_grafico.0.avancado }}%) {% endif %} , enquanto em {{ dados_grafico.1.disciplina }} o percentual de alunos abaixo do básico é de {{ dados_grafico.1.abaixo_basico }}%, indicando a necessidade de intervenções pedagógicas focadas. {% else %} Observe as diferenças entre as disciplinas para planejar ações de melhoria. {% endif %}
